# HR Dashboard

This is the HR Dashboard project. It includes the following pages:

- **Home Page**: `index.html`
- **Login Page**: `login.html`
- **Sign Up Page**: `signup.html`
- **About Us Page**: `about.html`
- **Company Information Page**: `company.html`

## Assets

- **CSS**: Custom styles are located in `assets/css/style.css`.
- **JavaScript**: Optional scripts are located in `assets/js/main.js`.
- **Images**: Images are stored in `assets/img/`.

## How to Use

1. Open `index.html` in a browser to view the home page.
2. Navigate to other pages using the links or directly opening the respective HTML files.
